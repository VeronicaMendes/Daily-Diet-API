# Daily-Diet-API

Projeto desenvolvido como desafio do módulo Desenvolvimento Avançado com Flask do curso da Rocketseat.

Essa API permite gerenciar o controle de dieta diária, possibilitando:

Registrar, editar, apagar e visualizar refeições;

Registrar detalhes como nome, descrição, data e hora, e se a refeição está dentro da dieta;

Armazenar as informações em banco de dados para persistência;

Este projeto tem o objetivo de reforçar conceitos avançados de desenvolvimento com Flask, além de estimular a autonomia na criação e estruturação de APIs
